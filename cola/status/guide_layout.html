폰트 스타일 주의사항 : 다음과 같이 폰트를 사용한다.
  • Noto Sans Korean를 기본으로 사용한다. (영문의 경우 Noto Sans를 사용)
  • 다른 폰트를 사용해야할 경우 이미지를 사용하며 적용시킨다.
    ※이미지 안 텍스트 내용이 길 경우 class="blind" 추가하여 이미지 아래에 따로 적어준다.

span class=""

class=""

class=""

테마 색상

상황을 설명할때, 기본적인 텍스트입니다.

class="" : 투명도 있는, 희미하게 지나가는 상황일 경우 나타나는 텍스트입니다.

class="" : 긍정적인 효과를 나타날때 나타나는 텍스트입니다.

class="" : 무언가를 발견하거나, 힌트 또는 팁을 받을 때 나타나는 텍스트입니다.

class="" : 위험한 상황, 혹은 불길한 무언가일 경우 텍스트입니다.
